Принудительное время открытия (A266.2)

Означает время (в секундах) принудительного открытия регулирующего клапана с электроприводом при обнаружении нагрузки (датчик включен). Данная функция компенсирует задержку перед тем, как изменение температуры будет зарегистрировано датчиком температуры теплоносителя. (Применимо только для приложения A266.2)

PNU: 12094
