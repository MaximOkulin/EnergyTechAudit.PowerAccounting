Принудительное время закрытия (A266.2)

Отвечает за время (в секундах) принудительного закрытия регулирующего клапана с электроприводом при прекращении нагрузки (датчик потока выключен). Данная функция позволяет компенсировать задержку перед тем, как изменение температуры будет зарегистрировано датчиком температуры теплоносителя. (Применимо только для приложения A266.2)

PNU: 12095
